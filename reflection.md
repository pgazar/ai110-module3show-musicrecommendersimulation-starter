# Recommendation Comparisons

- High-Energy Pop vs Chill Lofi: The pop profile pushes bright, danceable songs like `Sunrise City`, while the lofi profile shifts toward lower-energy, more acoustic songs like `Library Rain` and `Midnight Coding`. This makes sense because the genre, mood, energy, and acousticness targets move in opposite directions.
- High-Energy Pop vs Deep Intense Rock: Both profiles like high energy, so some strong tracks stay competitive, but the rock profile pulls toward heavier and faster songs like `Storm Runner` instead of upbeat dance-friendly pop. The shared energy preference explains overlap, while genre and mood explain the split.
- High-Energy Pop vs Conflicting Edge Case: The pop profile rewards happy and danceable tracks, while the conflicting profile produces a stranger mix because it asks for lofi plus very high energy and a mood the dataset does not contain. That mismatch shows how one strong category can dominate when the rest of the profile pulls in a different direction.
- Chill Lofi vs Deep Intense Rock: The lofi profile favors soft, acoustic, slow songs, while the rock profile favors fast, intense tracks with lower acousticness. The rankings separate cleanly because almost every major feature target is different.
- Chill Lofi vs Conflicting Edge Case: Both profiles mention lofi, so some of the same songs appear, but their order changes because the conflicting profile asks for much higher energy and much lower valence. This is a good example of genre overlap not being enough to make two users feel identical.
- Deep Intense Rock vs Conflicting Edge Case: Both profiles lean high-energy, so songs like `Storm Runner` and `Iron Anthem` remain relevant, but the edge-case profile still keeps some lofi songs near the top because the genre bonus is large. That comparison highlights one of the system's clearest biases.
